  <p>
   给定一个整数数组
   <em>
    nums
   </em>
   ，求出数组从索引
   <em>
    i
   </em>
   到
   <em>
    j
   </em>
   (
   <em>
    i
   </em>
   ≤
   <em>
    j
   </em>
   ) 范围内元素的总和，包含
   <em>
    i,  j
   </em>
   两点。
  </p>
  <p>
   <strong>
    示例：
   </strong>
  </p>
  <pre>给定 nums = [-2, 0, 3, -5, 2, -1]，求和函数为 sumRange()

sumRange(0, 2) -&gt; 1
sumRange(2, 5) -&gt; -1
sumRange(0, 5) -&gt; -3</pre>
  <p>
   <strong>
    说明:
   </strong>
  </p>
  <ol>
   <li>
    你可以假设数组不可变。
   </li>
   <li>
    会多次调用
    <em>
     sumRange
    </em>
    方法。
   </li>
  </ol>
