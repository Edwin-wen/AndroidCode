  <p>
   有
   <code>
    N
   </code>
   个网络节点，标记为
   <code>
    1
   </code>
   到
   <code>
    N
   </code>
   。
  </p>
  <p>
   给定一个列表
   <code>
    times
   </code>
   ，表示信号经过
   <strong>
    有向
   </strong>
   边的传递时间。
   <code>
    times[i] = (u, v, w)
   </code>
   ，其中
   <code>
    u
   </code>
   是源节点，
   <code>
    v
   </code>
   是目标节点，
   <code>
    w
   </code>
   是一个信号从源节点传递到目标节点的时间。
  </p>
  <p>
   现在，我们向当前的节点
   <code>
    K
   </code>
   发送了一个信号。需要多久才能使所有节点都收到信号？如果不能使所有节点收到信号，返回
   <code>
    -1
   </code>
   。
  </p>
  <p>
   <strong>
    注意:
   </strong>
  </p>
  <ol>
   <li>
    <code>
     N
    </code>
    的范围在
    <code>
     [1, 100]
    </code>
    之间。
   </li>
   <li>
    <code>
     K
    </code>
    的范围在
    <code>
     [1, N]
    </code>
    之间。
   </li>
   <li>
    <code>
     times
    </code>
    的长度在
    <code>
     [1, 6000]
    </code>
    之间。
   </li>
   <li>
    所有的边
    <code>
     times[i] = (u, v, w)
    </code>
    都有
    <code>
     1 &lt;= u, v &lt;= N
    </code>
    且
    <code>
     1 &lt;= w &lt;= 100
    </code>
    。
   </li>
  </ol>
