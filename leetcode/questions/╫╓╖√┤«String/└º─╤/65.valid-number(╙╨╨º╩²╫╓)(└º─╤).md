  <p>
   验证给定的字符串是否为数字。
  </p>
  <p>
   例如:
   <br/>
   <code>
    "0"
   </code>
   =&gt;
   <code>
    true
   </code>
   <br/>
   <code>
    " 0.1 "
   </code>
   =&gt;
   <code>
    true
   </code>
   <br/>
   <code>
    "abc"
   </code>
   =&gt;
   <code>
    false
   </code>
   <br/>
   <code>
    "1 a"
   </code>
   =&gt;
   <code>
    false
   </code>
   <br/>
   <code>
    "2e10"
   </code>
   =&gt;
   <code>
    true
   </code>
  </p>

    说明:
