  <p>
   给定一个字符串，你需要反转字符串中每个单词的字符顺序，同时仍保留空格和单词的初始顺序。
  </p>
  <p>
   <strong>
    示例 1:
   </strong>
  </p>
  <pre>
输入: "Let's take LeetCode contest"
输出: "s'teL ekat edoCteeL tsetnoc"<strong><strong><strong> </strong></strong></strong>
</pre>
  <p>
   <strong>
    <strong>
     <strong>
      <strong>
       注意：
      </strong>
     </strong>
    </strong>
   </strong>
   在字符串中，每个单词由单个空格分隔，并且字符串中不会有任何额外的空格。
  </p>
