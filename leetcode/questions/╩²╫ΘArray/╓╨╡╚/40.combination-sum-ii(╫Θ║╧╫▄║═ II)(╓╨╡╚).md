  <p>
   给定一个数组
   <code>
    candidates
   </code>
   和一个目标数
   <code>
    target
   </code>
   ，找出
   <code>
    candidates
   </code>
   中所有可以使数字和为
   <code>
    target
   </code>
   的组合。
  </p>
  <p>
   <code>
    candidates
   </code>
   中的每个数字在每个组合中只能使用一次。
  </p>
  <p>
   <strong>
    说明：
   </strong>
  </p>
  <ul>
   <li>
    所有数字（包括目标数）都是正整数。
   </li>
   <li>
    解集不能包含重复的组合。
   </li>
  </ul>
  <p>
   <strong>
    示例 1:
   </strong>
  </p>
  <pre><strong>输入:</strong> candidates = <code>[10,1,2,7,6,1,5]</code>, target = <code>8</code>,
<strong>所求解集为:</strong>
[
  [1, 7],
  [1, 2, 5],
  [2, 6],
  [1, 1, 6]
]
</pre>
  <p>
   <strong>
    示例 2:
   </strong>
  </p>
  <pre><strong>输入:</strong> candidates = [2,5,2,1,2], target = 5,
<strong>所求解集为:</strong>
[
  [1,2,2],
  [5]
]</pre>
