  <p>
   给定一个元素都是正整数的数组
   <code>
    A
   </code>
   ，正整数
   <code>
    L
   </code>
   以及
   <code>
    R
   </code>
   (
   <code>
    L &lt;= R
   </code>
   )。
  </p>
  <p>
   求连续、非空且其中最大元素满足大于等于
   <code>
    L
   </code>
   小于等于
   <code>
    R
   </code>
   的子数组个数。
  </p>
  <pre><strong>例如 :</strong>
<strong>输入:</strong> 
A = [2, 1, 4, 3]
L = 2
R = 3
<strong>输出:</strong> 3
<strong>解释:</strong> 满足条件的子数组: [2], [2, 1], [3].
</pre>
  <p>
   <strong>
    注意:
   </strong>
  </p>
  <ul>
   <li>
    L, R  和
    <code>
     A[i]
    </code>
    都是整数，范围在
    <code>
     [0, 10^9]
    </code>
    。
   </li>
   <li>
    数组
    <code>
     A
    </code>
    的长度范围在
    <code>
     [1, 50000]
    </code>
    。
   </li>
  </ul>
